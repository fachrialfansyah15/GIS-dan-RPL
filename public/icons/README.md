# Marker Icons

## Cara Generate PNG Icons:

1. Buka file `generate-icons.html` di browser
2. Klik tombol "Download PNG" untuk setiap ikon
3. Simpan file dengan nama:
   - marker-proses.png
   - marker-selesai.png

## Atau gunakan SVG langsung:

File SVG sudah tersedia dan bisa digunakan langsung di Leaflet:
- marker-proses.svg (biru dengan ikon wrench)
- marker-selesai.svg (hijau dengan ikon check)

## Spesifikasi:

- **marker-proses.png**: Biru (#3b82f6) dengan ikon wrench
- **marker-selesai.png**: Hijau (#10b981) dengan ikon check mark
- Ukuran: 40x40 px
- Format: PNG dengan transparansi
